   Wraps executeAgentTool with extension-shaped defaults. Signature:                                           
                                                                                                               
   ```ts                                                                                                       
   forkAgent(opts: {                                                                                           
     prompt: string;              // first user message to the forked agent

     model?: string;              // default: parent's model — required for cache hit                          
     signal?: AbortSignal;                                                                                     
     description?: string;        // for logs/UI

     handle: AgentHandle;         // .wait(), .abort(), .status                                                
     sessionId: string;                                                                                        
   }>;                                                                                                         
 ```                                                                                                           
                                                                                                               
 Must:                                                                                                         
 - Run in background (returns immediately, doesn't block the hook).                                            
 - Use the parent's frozen system prompt for cache preservation (look at how                                   
   executor.ts handles context: "fork" + parentSystemPrompt today).                                            
 - Inherit cwd from ctx.                                                                                       
 - Respect ctx.signal for abort propagation.                                                                   
 - Default model = parent's model. Different model = cache miss; document this.                                
 - allowedTools restriction enforced at tool-dispatch time.
